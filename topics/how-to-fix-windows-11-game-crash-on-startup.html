Why Games Crash on Startup in Windows 11

Few things are more frustrating than clicking "Play" only to watch your game crash before reaching the main menu. It feels like buying a pizza and discovering the box contains only the receipt.

In many cases, startup crashes occur because Windows system files become corrupted, game files are damaged, temporary files interfere with the launch process, or essential Windows services stop functioning correctly. Fortunately, Windows 11 includes several built-in repair tools that can help you fix these issues without reinstalling the entire operating system.

This guide walks you through the most effective methods to repair corrupted files and stop games from crashing immediately after launch.

Quick Overview of the Main Fixes

Fix Purpose Difficulty Success Rate
SFC Scan Repairs corrupted Windows system files Easy High
DISM Repair Repairs damaged Windows image files Easy Very High
Verify Game Files Repairs corrupted game installations Easy Very High
Clear Temp Files Removes damaged cache and temporary files Easy Moderate

Run SFC /Scannow to Repair Corrupted System Files

The System File Checker (SFC) scans Windows for missing or damaged system files and automatically replaces them with healthy copies.

To run SFC:

  1. Press Windows + X.
  2. Select Windows Terminal (Admin).
  3. Click Yes if prompted by User Account Control.
  4. Enter the following command:

sfc /scannow

Press Enter and wait patiently. Depending on your system, the scan may take several minutes.

If SFC finds corrupted files, it will attempt to repair them automatically. Once the scan completes, restart your PC and test the game again.

Think of SFC as Windows' repair technician. It walks through the operating system looking for broken parts and quietly replaces them while you resist the urge to mash random keys.

Use DISM for Deeper Windows Repair

If SFC cannot repair all damaged files, the Deployment Image Servicing and Management tool (DISM) can repair the Windows image itself.

Open Windows Terminal as Administrator and run the following commands one at a time:

DISM /Online /Cleanup-Image /CheckHealth

DISM /Online /Cleanup-Image /ScanHealth

DISM /Online /Cleanup-Image /RestoreHealth

The final command may take some time because Windows downloads replacement files when necessary.

After DISM finishes:

  1. Restart your computer.
  2. Run SFC again.
  3. Launch your game.

Many stubborn game crashes disappear after a successful DISM repair because Windows regains access to critical components required by game launchers and graphics services.

Verify Game Files Through Your Launcher

Even if Windows is healthy, your game installation might not be. Corrupted game files commonly occur after interrupted downloads, failed updates, storage errors, or unexpected shutdowns.

Steam

  1. Open Steam.
  2. Go to Library.
  3. Right-click the game.
  4. Select Properties.
  5. Open Installed Files.
  6. Click Verify Integrity of Game Files.

Epic Games Launcher

  1. Open Epic Games Launcher.
  2. Go to Library.
  3. Click the three dots beside the game.
  4. Select Manage.
  5. Click Verify.

Xbox App

  1. Open Xbox App.
  2. Select the game.
  3. Click Manage.
  4. Choose Files.
  5. Select Verify and Repair.

The launcher will compare installed files against official versions and download any missing or damaged data.

This process often fixes startup crashes caused by corrupted textures, broken configuration files, or incomplete updates.

Clear Temporary Files and Cache

Temporary files occasionally become corrupted and interfere with game launches.

To clear temporary files:

  1. Press Windows + R.
  2. Type %temp% and press Enter.
  3. Delete all files inside the folder.
  4. Skip any files currently in use.

Next:

  1. Press Windows + R.
  2. Type temp and press Enter.
  3. Delete the contents.

You can also clear Prefetch files:

  1. Press Windows + R.
  2. Type prefetch.
  3. Delete the files inside.

Don't worry. Windows automatically rebuilds these files when necessary.

Think of this as cleaning your gaming desk. The game may not run faster immediately, but removing clutter often solves strange behavior.

Re-register Windows Apps Using PowerShell

Some games rely on Microsoft Store components, Xbox services, or Windows apps. If these become damaged, startup crashes can occur.

Open PowerShell as Administrator and run:

Get-AppxPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

This command re-registers Windows applications and restores broken app registrations.

The process may display several messages while running. Allow it to finish completely before restarting your computer.

Additional Tips if the Game Still Crashes

If the issue persists after repairing files, consider these additional troubleshooting steps:

  • Update your graphics drivers.
  • Install all available Windows updates.
  • Disable overlays from Discord, Steam, and Xbox Game Bar.
  • Temporarily disable third-party antivirus software.
  • Run the game as Administrator.
  • Check available storage space.
  • Reinstall the affected game.

A game cannot launch properly if critical files, drivers, or services remain unavailable.

Final Thoughts

When a game crashes immediately after startup, corrupted files are often the hidden culprit. Fortunately, Windows 11 provides powerful repair tools that can restore damaged system components without requiring a full reinstall.

Start with sfc /scannow, follow up with DISM if necessary, verify your game files through the launcher, and clear temporary files that may be causing conflicts. If Microsoft Store or Xbox-related games continue crashing, re-registering Windows apps can often resolve the issue.

In many cases, these repairs take less time than reinstalling a 100 GB game. Your internet connection will appreciate the break, and you'll get back to gaming much faster.
